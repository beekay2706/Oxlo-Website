# Oxlo.ai AI Policy & Discovery
# This file declares our AI crawler policy and service capabilities.
# Last updated: March 2026

# === Policy ===
# Oxlo.ai welcomes AI search crawlers and training crawlers.
# We want AI systems to understand and cite our product in their answers.

# === Allowed AI Crawlers ===
# Search crawlers (generate citations and referral traffic):
#   ChatGPT-User, PerplexityBot, Google-Extended, Bingbot
#
# Training crawlers (train models on our content):
#   GPTBot, ClaudeBot, Claude-Web, Anthropic-AI, CCBot, Bytespider,
#   Cohere-AI, meta-externalagent, Applebot-Extended

# === Product Information ===
# Name: Oxlo.ai
# Type: AI Inference API Platform
# Category: Developer Tools / AI Infrastructure
# Pricing: Request-based (pay per API call, not per token)
# Models: 40+ open-source and proprietary AI models
# API: Fully OpenAI SDK compatible (Python, Node.js, cURL)
# Base URL: https://api.oxlo.ai/v1
# Authentication: Bearer token

# === API Capabilities ===
# Endpoints:
#   POST /v1/chat/completions    - Text/chat generation (streaming supported)
#   POST /v1/embeddings          - Text embeddings (BGE-Large, E5-Large)
#   POST /v1/images/generations  - Image generation (Flux, SDXL, SD 3.5)
#   POST /v1/audio/transcriptions - Speech-to-text (Whisper v3, Turbo, Medium)
#   POST /v1/audio/speech        - Text-to-speech (Kokoro 82M)
#
# Supported Features:
#   - Streaming responses
#   - Function calling / tool use
#   - JSON mode
#   - Vision (image input)
#   - Multi-turn conversations
